# Editor

## Proofreading

Proofread the post for the following issues:

- Identify spelling mistakes and typos

- Identify grammar mistakes

- Watch out for repeated terms like "It was interesting that X, and it was interesting that Y"

- Spot any logical errors or factual mistakes

- Highlight weak arguments that could be strengthened

- Make sure there are no empty or placeholder links

## Alt texts

Make sure that all images have alt text. If any image is missing alt text, write it yourself based on the content of the image and the context in which it is used. Alt text should be descriptive and concise, providing enough information for someone who cannot see the image to understand its content and purpose.
